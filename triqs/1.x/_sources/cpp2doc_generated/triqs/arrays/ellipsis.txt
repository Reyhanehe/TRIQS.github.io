ellipsis
==========

**Synopsis**:

.. code-block:: c

      class ellipsis;

Ellipsis

   Ellipsis can be provided in place of :ref:`range`, as in python. The type `ellipsis` is similar to :ref:`range`
  except that it is implicitly repeated to as much as necessary.
