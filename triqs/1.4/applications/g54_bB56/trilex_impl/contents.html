This repository contains functions pertaining to the local vertex \(\Lambda^\eta_{uv\alpha}(i\omega,i\Omega)\).

There are three main groups of functions:
  • “lattice” functions which comprise the calculation of the main TRILEX formulae \(\Sigma(k) \sim \lambda G(k+q) W(q) \Lambda\) (sigma_gwl) and \(P(q) \sim \lambda G(k+q)G(k) \Lambda\) (p_ggl); the computation of the \(\zeta\) factor is also implemented here (compute_zeta)
  • “local” functions which comprise the calculation of the vertex starting from the full three-point correlation function (and the legs, compute_Lambda), as well as the implementation of the exact formulae \(\Sigma \sim \lambda G W \Lambda\) (sigma_gwl) and \(P \sim \lambda GG \Lambda\) (p_ggl) at the local level. These functions are generic (can be used beyond single-band, single-site, normal-state TRILEX)
  • functions specific to single-site TRILEX applied to the Hubbard model (compute_Lambda_single_band_Hubbard, evaluate_vertex)...
