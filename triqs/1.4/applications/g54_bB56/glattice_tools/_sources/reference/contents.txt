Reference documentation
=========================

This repository contains functions pertaining to :math:`G(\mathbf{k},i\omega)`, any function :math:`f(\mathbf{k})` on a mesh, and the underlying objects (mesh on Brillouin zone, Brillouin zone domain...).

The c++ (resp. python) source code is in the folder **/c++** (resp. **/python**).
Should some item not be documented, an useful place to look for examples is the folder **/test**.
