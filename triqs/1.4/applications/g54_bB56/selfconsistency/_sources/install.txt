Installation
=========================

You need to install the following elements:

   #. TRIQS (**master**)
   #. adaptative_integration (**master**) 
   #. cthyb_spin or ctint or cthyb_segment
   #. trilex_impl (**master**)

A script to do all this is provided in the source : **/install.sh**.
