is_gf_real_in_tau¶

Synopsis:

 template<typename T, typename S, typename E>
 bool is_gf_real_in_tau (gf_const_view<triqs::gfs::imfreq, T, S, E> g, double tolerance)
;

#include <None>
Test if gf is real in tau

Parameters¶

  • g: the gf
  • tolerance: tolerance threshold \(\epsilon\)

Return value¶

true iif $$forall n,; max_{ab}|g^*_{ab}(-iomega_n)-g_{ab}(iomega_n)|<epsilon$$
