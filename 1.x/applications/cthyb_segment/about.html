Authors¶

The continuous-time hybridization-expansion segment solver has been written by T. Ayral, H. Hafermann, M. Ferrero and O. Parcollet. It is based on a hybridization expansion of the partition function as described in references [#ctqmc1]_ and [#ctqmc2]_. The computation of the imaginary-time Green’s function has been improved with the use of Legendre polynomials following reference [#ctqmc3]_.

License¶

The segment solver is private at the moment. Please contact the authors for more information.
