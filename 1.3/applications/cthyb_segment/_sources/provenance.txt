Provenance
=================


Here is how to print out the hash of the git commit you are using (together with the corresponding TRIQS commit)

.. runblock:: python

    from pytriqs.applications.impurity_solvers.cthyb_segment.version import *

    show_git_hash()
    show_version()

    print "triqs_hash: ", triqs_hash
    print "cthyb_hash: ", cthyb_hash
