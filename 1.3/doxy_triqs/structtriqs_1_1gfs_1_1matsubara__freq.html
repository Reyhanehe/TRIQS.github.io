Detailed Description

A matsubara frequency, i.e.

  • n : int, the index
  • beta : double, the temperature inverse
  • statistic : Fermion or Boson

Can be casted into a complex.

  • Every operations is done by casting to complex, except addition and substraction of matsubara_freq, which return matsubara_freq and work on the index
